
# Marketing ROI Analysis – Simple Linear Regression

## Project Overview
This project uses Python and statsmodels to analyze marketing data and build a Simple Linear Regression model to predict Sales.

## Files Included
- regression_analysis.ipynb
- README.md
- marketing dataset CSV file

## Technologies Used
- Python
- pandas
- matplotlib
- seaborn
- statsmodels
- scipy

## Environment Setup

Install dependencies:

```bash
pip install pandas matplotlib seaborn statsmodels scipy notebook
```

## Project Workflow
1. Load and clean dataset
2. Perform exploratory data analysis
3. Identify strongest predictor
4. Build OLS regression model
5. Validate assumptions using diagnostic plots
6. Interpret business insights

## Expected Outcome
The analysis identifies the marketing channel with the strongest ROI impact on Sales and provides data-driven recommendations for budget allocation.
